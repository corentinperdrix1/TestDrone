# Exercice

1. Contexte
Pour cet exercice, j'ai voulu faire un drone qui peut avancer, reculer, tourner à droite et à gauche, monter et descendre dans un environnement.

2. Mouvement
Avec ZQSD, ESPACE et CTRL GAUCHE.
Le drone peut aussi tourner en déplaçant la souris à gauche ou à droite. Monter ou descendre la souris permet d'incliner le drone vers l'avant et l'arrière.
Pour faire les mouvements, je me suis aidé de cette vidéo : https://www.youtube.com/watch?v=Z9zEEY7dGaM.
Et après, j'ai rajouté MOVEUP pour monter et descendre.

3. Map
Pour la map, j'ai téléchargé une Map sur Fab que j'ai importée dans mon projet.


4. Procédure pour le lancement du projet
Quand vous lancez le projet, vous allez dans le tiroir à contenu et, dans « contenu », vous verrez ROAD, vous double cliquez dessus pour le charger. Après ça, vous aurez juste besoin de lancer la simulation.
